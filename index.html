HTML

HTML is a standard markup language for creating Web pages.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.



Doctype

Doctype is used to define the document type.



HTML Elements

Html tag

HTML elements are the building blocks of HTML pages.


Head tag

The head element contains information about the HTML document.


Meta Meta tags are used to specify the page's metadata.
Title The title element defines the title of the HTML document.


Body tag

The body element represents the visible content shown to the user.
